========================================================================
                         QUICK START GUIDE
========================================================================

STEP 1: SETUP (First Time Only)
   Double-click: 1-setup.bat
   This will:
   - Create Python virtual environment
   - Install all Python packages
   - Install all Node.js packages
   - Train the ML model

STEP 2: START APPLICATION
   Double-click: 2-start.bat
   This will:
   - Start backend on http://127.0.0.1:5000
   - Start frontend on http://localhost:5173
   - Open browser automatically

STEP 3: STOP APPLICATION
   Double-click: 3-stop.bat
   This will stop all running servers

========================================================================
                         TROUBLESHOOTING
========================================================================

ISSUE: Scikit-learn version warnings
FIX: Run backend/retrain-model.bat

ISSUE: Port already in use
FIX: Run 3-stop.bat, wait 5 seconds, then run 2-start.bat

ISSUE: Setup fails
FIX: Ensure Python and Node.js are installed
     Run as Administrator if permission errors occur
